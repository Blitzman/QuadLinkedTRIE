Detailed Description

template<class T>
class QuadLinkedTrieNode< T >

Single node for a Quad Linked TRIE data structure.

This class represents a quad linked TRIE single node which holds a character key and a certain type value for that node. Moreover, the node contains some references to its parent, the left and right brothers and its first child. In addition, the node can be marked as an actual word in the data structure.

The node offers some functionality to navigate through other related nodes and also obtaining and modifying its value, key and word marker.
